# BlinkCode

BlinkCode is an interactive Morse code decoder that translates blink/tap inputs into readable text in real time using a binary tree (trie) structure.

## 🚀 Features
- Press-based input for dot (·) and dash (–)
- Real-time decoding
- Live path tracing through the Morse tree
- Visual feedback for each signal
- Clean and minimal UI

## 🧠 How it Works
The system uses a binary tree where:
- Left branch represents a dot (·)
- Right branch represents a dash (–)

Each input navigates the tree until a valid character is reached.
